# Pact — landing page

The marketing site for **Pact**, a strict, honest website blocker built as a commitment device.

Static HTML/CSS, no build step. Served via GitHub Pages.

- `index.html` — English
- `es/` — Spanish
- `zh/` — Simplified Chinese
- `style.css` — shared styles

> Pact is a commitment device, not a lock. You can remove it anytime from your browser's
> extensions page. The site never claims otherwise.
